# Projeto Formulário de Matrícula

Este projeto foi desenvolvido como parte de um desafio do curso da Rocketseat, com o objetivo de praticar e consolidar conhecimentos em desenvolvimento web moderno.

## Sobre o projeto
O sistema consiste em um formulário de matrícula escolar, com layout responsivo e foco em experiência do usuário. O desafio envolveu criar uma interface intuitiva, estilizada e funcional, utilizando boas práticas de HTML e CSS.

## Tecnologias utilizadas
- **HTML5**
- **CSS3** (incluindo variáveis CSS, grid, flexbox, transições)

## Estrutura de pastas
```
├── index.html
├── styles/
│   ├── global.css
│   └── index.css
├── assets/
│   └── icons/
```

## Como executar
Abra o arquivo `index.html` em seu navegador. Não é necessário backend ou dependências externas para rodar o projeto.
